# BakBak

Anonymous chat and video sharing platform.

## About
BakBak is a stranger chat platform inspired by Omegle, offering anonymous and unfiltered conversations with random people worldwide.

**Core Principles:**
- No accounts required
- No chat history saved
- No tracking or data collection
- Completely anonymous interactions

## Features
- **Text Chat**: Connect with random strangers for text-based conversations
- **Video Sharing**: Share video content anonymously
- **Instant Matching**: Get connected immediately, 24/7
- **Moderation System**: AI-powered text and video moderation for safety
- **Zero Privacy Invasion**: No personal data collection or storage

## Privacy
- No registration or login required
- No IP logging
- No conversation storage
- Complete anonymity guaranteed
 
## Tech Stack
- **Frontend**: HTML5, CSS3, JavaScript
- **Backend**: WebSocket, WebRTC
- **Moderation**: Perspective API (text), NSFWJS (video)
